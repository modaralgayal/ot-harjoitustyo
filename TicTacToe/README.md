# TicTacToe

Sovellus on Ristinolla peli jota pelaa kaksi pelaajaa paikallisesti yhdellä koneella. Sovellusta on tarokitus myös laajentaa isomman kokoiseen ristinolla peliin, eli 4x4, 5x5 jne.
